# Jungle-Room-Cafe-HTML-CSS

This website is related to a hypothetical Café and cocktail bar in Douala, Cameroon. The site received inspiration from a real
Café in Oslo, Norway: [https://www.cafesor.no/]
Link: [https://github.com/atchakounte/Jungle-Room-Cafe-HTML-CSS](https://github.com/atchakounte/Jungle-Room-Cafe-HTML-CSS)

## User stories

AS A person
I WANT TO access the Café's location
SO THAT simply, I have place to belong

AS A student
I WANT TO verify if the Café provides access to the Internet
SO THAT I have a place to study

AS A freelancing self-employed professional
I WANT TO book a table for a meeting space
SO THAT I have an informal environment for first-time client introductions

AS A young partygoer
I WANT TO verify the cocktail bar's hours
SO THAT I know when the nightclub hours are open

AS A young partygoer
I WANT TO access a list of the Café's events
SO THAT I know when events have a DJ and music genre
